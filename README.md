# sudokusolver

A Sudoku solver application written in Rust.

## Requirements

- Rust 1.92.0 or later

## Building

To build the project:

```bash
cargo build
```

## Running

To run the application:

```bash
cargo run
```

## Testing

To run tests:

```bash
cargo test
```

## Development

This project uses Rust's latest stable edition (2024).
